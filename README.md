# 📊 Power BI Learning Repository  

Welcome to my **Power BI repository**!  
This project contains a structured collection of **practice files, datasets, and learning materials** organized day-wise, covering data visualization, transformation, modeling, and analytics using Power BI.

---

## 📂 Repository Structure  

### 📘 Day 1 – Basics & Visualizations  
- Introduction to Power BI visuals  
- Practice activity file  
- Dataset: Excel-based data  
- `.pbix` file for basic report creation  

### 🛠️ Day 2 – Data Transformation & Dashboards  
- Data cleaning and transformation  
- HR analytics dashboard projects  
- Bank dataset analysis  
- Use of images and UI elements in dashboards  
- Multiple `.pbix` dashboard files  

### 🔄 Day 3 – Query Operations  
- Append and Merge queries  
- Sales data analysis  
- Supporting datasets (monthly sales, employee & salary data)  
- Documentation PDFs explaining transformations  

### 📊 Day 4 – DAX & Data Modeling  
- DAX expressions and calculations  
- Data modeling techniques  
- Append queries with multiple data sources  
- Hands-on `.pbix` files and CSV datasets  

### ⚡ Day 5 – Schema Design  
- Star schema and Snowflake schema concepts  
- Data cardinality and relationships  
- Supporting PDFs, PPTs, and diagrams  

### 🚀 Day 6 – Advanced Integrations  
- Python integration in Power BI  
- SQL embedding  
- Advanced analytics workflows  
- Supporting datasets and scripts
